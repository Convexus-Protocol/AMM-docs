# Installation

To interact with the SDK - we recommend installing though the npm package

```javascript
npm i --save-dev @convexus/sdk
npm i --save-dev @convexus/sdk-core
npm i --save-dev @convexus/icon-toolkit
```

# Usage

```javascript
import { Pool } from '@convexus/sdk'
```
